About AgriSmart

AgriSmart is an AI-powered agricultural advisory system that helps farmers make data-driven decisions through multiple ML/DL modules.

Project Goals
  • Predict crop yield using environmental inputs.
  • Detect common plant diseases from leaf images.
  • Recommend fertilizer (NPK) based on crop, soil and stage.
  • Forecast rainfall patterns and provide advisory guidance.
AgriSmart Modules
  • Crop Yield Prediction — ML regression demo
  • Plant Disease Detection — CNN classification demo
  • Fertilizer Recommendation — rule-based NPK recommendation demo
  • Weather Pattern Analysis — rainfall forecast demo
Architecture Snapshot
Input → Model(s) → Output → Advisory
  • Yield: rainfall, pesticide usage, temperature
  • Disease: leaf image
  • Fertilizer: crop type, soil type, growth stage, rainfall
  • Weather: year-based rainfall forecast
