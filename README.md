# 🌱 AgriSmart – AI-Powered Smart Agriculture Platform

## 📖 Overview

AgriSmart is an AI-powered smart agriculture platform that helps farmers make data-driven decisions using Machine Learning, Artificial Intelligence, and real-time agricultural insights. The system assists in crop selection, disease detection, yield prediction, fertilizer recommendations, and intelligent farming guidance to improve productivity and sustainability.

---

## 🎯 Problem Statement

Farmers often face challenges such as:

- Unpredictable weather conditions
- Crop diseases and pest attacks
- Improper fertilizer usage
- Low crop productivity
- Lack of expert agricultural guidance

AgriSmart addresses these challenges by providing AI-driven recommendations and predictive analytics to support smarter farming decisions.

---

## ✨ Key Features

### 🌾 Crop Recommendation
Recommends the most suitable crops based on:
- Soil nutrients (N, P, K)
- Temperature
- Humidity
- Rainfall
- Soil pH

### 🦠 Plant Disease Detection
- Detects plant diseases from leaf images
- Uses Deep Learning models for classification
- Provides disease information and preventive measures

### 📈 Yield Prediction
- Predicts crop yield using historical and environmental data
- Helps farmers plan production effectively

### 🧪 Fertilizer Recommendation
- Suggests appropriate fertilizers based on soil conditions
- Supports efficient nutrient management

### 🌦 Weather Intelligence
- Provides weather forecasts and climate insights
- Assists in irrigation and cultivation planning

### 🤖 AI Farming Assistant
- LLM-powered agricultural chatbot
- Answers farming-related queries
- Provides intelligent recommendations
