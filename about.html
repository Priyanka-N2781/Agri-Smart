Project Overview

What is AgriSmart?

AgriSmart is a comprehensive, AI-driven agricultural advisory web application that empowers farmers, agri-researchers, and agricultural students with smart, data-backed insights. It combines machine learning, deep learning, and real-time data analysis into a single, seamless platform.

From predicting how much crop a field will yield based on environmental conditions, to scanning a leaf photo and identifying which of 33 different diseases might be affecting it — AgriSmart covers the full spectrum of modern precision agriculture.

Objectives

Project Goals

Predict crop yield accurately using environmental parameters and ML regression.

Detect plant diseases from leaf images using CNN computer vision.

Recommend optimal NPK fertilizer ratios for given crops and soil conditions.

Forecast annual rainfall using historical time-series regression.

Expose all features via a REST API for integration with IoT and mobile apps.

Demonstrate a full-stack ML web app with professional UI and documentation.
