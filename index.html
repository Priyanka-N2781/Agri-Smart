Workflow

How AgriSmart Works

1. Input Data

Enter your field parameters — crop type, soil info, weather data, or upload a leaf photo.

2. AI Processing

Our trained ML models analyse the data using Random Forest, CNN, or Linear Regression.

3. Smart Results

Receive instant predictions, disease diagnoses, NPK recommendations, or rainfall forecasts.

4. Take Action

Follow the tailored advisory to optimise yield, prevent disease, and plan irrigation.

POST /api/predict-yield 
{
  "crop": "Wheat",
  "rainfall_mm": 820,
  "pesticides_tonnes": 45,
  "avg_temp_c": 23.5
}

// Response
{
  "success": true,
  "data": {

  }
}
